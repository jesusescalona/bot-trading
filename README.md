## INSTALACION

1) CLONAR EL REPOSITORIO
2) ENTRAR A LA CARPETA
3) ABRO CUALQUIERA DE LOS ARCHIVOS PYTHON E INSTALO LA EXTENSION QUE ME SUGIERE.
4) CTRL + SHIFT + P SI ESTOY EN WINDOWS
5) COMMAND + SHIFT + P SI ESTOY EN MACOS
6) SELECCIONO CREAR AMBIENTE O CREATE ENVIROMENT
7) SELECCIONO LA VERSIÓN DE PYTHON INSTALADA Y SI NO TENGO LA INSTALO
8) EL ENVIROMENT DEBE QUEDAR EN LA RAIZ DEL WORKSPACE PARA QUE PUEDA RECONOCER
LA ISNTALACIÓN DE PYTHON GLOBAL AL MOMENTO DE INSTALAR LAS LIBRERIAS.
9) UBICADO EN LA RAIZ DEL WORKSPACE EJECUTO: source .venv/bin/activate
ESTO ACTIVARA EL ENVIROMENT PARA PODER INSTALAR LAS LIBRERIAS, DE LO CONTRARIO NO LAS RECONOCERA.
9) INSTALO LAS DEPENDENCIAS O LIBRERIAS CON: 
pip install nombre_dependencias_librerias (Para Python 2)
pip3 install nombre_dependencias_librerias (Para Python de la versión 3 en adelante)
10) CORRO EL ARCHIVO CON EL ICONO QUE DEBE APARECER EN LA PARTE SUPERIOR DERECHA EN VS CODE.
