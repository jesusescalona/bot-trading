# Bot Trading - Fly.io Multi-Symbol

## USO DEL BOT DE TELEGRAM
Importante: cómo usarlo desde Telegram

/pause → no abre nuevas operaciones

/resume → vuelve a abrir

/status → te dice si está pausado + posición

/close → pide confirmación

/close yes → cierra a MARKET (reduceOnly)

## INSTALACION

1) CLONAR EL REPOSITORIO
2) ENTRAR A LA CARPETA
3) ABRO CUALQUIERA DE LOS ARCHIVOS PYTHON E INSTALO LA EXTENSION QUE ME SUGIERE.
4) CTRL + SHIFT + P SI ESTOY EN WINDOWS
5) COMMAND + SHIFT + P SI ESTOY EN MACOS
6) SELECCIONO CREAR AMBIENTE O CREATE ENVIROMENT
7) SELECCIONO LA VERSIÓN DE PYTHON INSTALADA Y SI NO TENGO LA INSTALO
8) LA CARPETA .vemv/ DEL ENVIROMENT DEBE QUEDAR EN LA RAIZ DEL WORKSPACE PARA QUE PUEDA RECONOCER
LA ISNTALACIÓN DE PYTHON GLOBAL AL MOMENTO DE INSTALAR LAS LIBRERIAS.
9) ME UBICO EN EN LA RAIZ DEL WORKSPACE EJECUTO: source .venv/bin/activate
ESTO ACTIVARA EL ENVIROMENT PARA PODER INSTALAR LAS LIBRERIAS, DE LO CONTRARIO NO LAS RECONOCERA.
10) LA EXPLICACIÓN ANTERIOR SOLO APLICA SI TENGO VARIOS PROYECTOS CARGADOS EN EL MISMO WORKSPACE.
11) SI UNICAMENTE CARGO EL PROYECTO DE LA CARPETA SOLO, NO TENDRÉ QUE HACER LOS PASO 8 Y 9
Y AUTOMATICAMENTE SE ME ACTUALIZARA LA TERMINAL INDICANDOME ENTRE PARENTESIS (.vemv)
ESTA ES LA SEÑAL QUE SE SETEO CORRECTAMENTE LA CONFIGURACIÓN DEL AMBIENTE AL PROYECTO.
12) INSTALO LAS DEPENDENCIAS O LIBRERIAS CON: 
pip install nombre_dependencias_librerias (Para Python 2)
pip3 install nombre_dependencias_librerias (Para Python de la versión 3 en adelante)
ESTE ES EL COMANDO PARA INSTALARLAS TODAS:
pip3 install pandas pyhthon-binance backtrader Matplotlib
13) EL NOMBRE DE LA VARIABLE dataname ubicada en el script 2 llamado "Carpeta 2.py"
DEBE SER IGUAL AL NOMBRE DEL ARCHIVO QUE SE GENERA EN EL script 1 llamado "Carpeta 1.py"
14) CORRO EL ARCHIVO CON EL ICONO QUE DEBE APARECER EN LA PARTE SUPERIOR DERECHA EN VS CODE.
